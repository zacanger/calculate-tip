# calculate-tip

Tiny command-line tip calculator.

--------

## Installation:

`npm i -g calculate-tip`

## Usage:

`calculate-tip price percentage`

`percentage` defaults to `20`.

Examples:

`calculate-tip 39.99`

`calculate-tip 39.99 17`
